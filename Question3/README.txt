To run:
Open "Run" (On Windows: Windows button + R)
and enter the following command:

chrome.exe --user-data-dir="C:/Chrome dev session" --disable-web-security

Then open the file index.html in the browser that appears. 

-------------------------------------------------------------------------

Video controls:
Play/Pause: This will play and pause the video.
Infinity sign: This will make it so that when the video finishes, it repeats the current video
Volume: This is used to turn up/down the volume - and to mute if clicking the sound icon.
HD: This does nothing.
Outward arrows: This displays all of the tags of the video.
CC: This displays the top tag of the video.
Resize icon: This will display the series that the video is a part of.


Known bugs: 
When clicking the timeline of the video above the infinity sign button or volume, it will
click the buttons instead of the timeline. This does not happen at any other point on the
timeline.
